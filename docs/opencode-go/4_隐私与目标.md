## 隐私保护

| 维度 | 说明 |
|:-----|:------|
| 目标用户 | 主要面向**国际用户** |
| 托管区域 | US（美国）、EU（欧盟）、Singapore（新加坡） |
| 数据使用 | **零保留策略**——提供商不会将你的数据用于模型训练 |
| 可用性 | 确保稳定的全球访问 |

## 项目目标

OpenCode Go 的四个核心目标：

### 1. 降低门槛

通过低成本订阅让更多人能够无门槛地使用 AI 编程。

- 首月 $5，之后 $10/月
- 无需按量付费，预算可控

### 2. 可靠访问

为最佳开源编程模型提供可靠的访问。

- 经过基准测试的模型和提供商组合
- 稳定的全球基础设施

### 3. 精选模型

精选经过测试和基准评估、适合编程 Agent 使用的模型。

- 不在列表中的模型可能质量参差不齐
- 列表会持续更新

### 4. 无锁定（No Lock-in）

允许与 OpenCode 一起使用任何其他提供商。

- 与 OpenCode 中其他提供商工作方式一致
- 随时可以切换回 Claude/GPT 等其他提供商
- API 密钥和配置完全可控

## 常见问题

### Q: 可以用在中国吗？
A: 服务主要面向国际用户，托管在 US/EU/Singapore。中国境内使用可能延迟较高。

### Q: 模型会更新吗？
A: 会。OpenCode 团队持续测试和添加新模型，列表可能随时变化。

### Q: 超出限制怎么办？
A: 可以在控制台中启用 **Use balance** 选项，超出限制后回退使用 Zen 余额。

### Q: 和直接调用 DeepSeek API 比怎么样？
A: Go 是聚合服务，提供了多个开源模型的统一访问入口。如果只用 DeepSeek，直接调用 DeepSeek 官方 API 可能更便宜。Go 适合需要**多个开源模型**且不想分别管理多个 API 密钥的场景。
