# OpenCode Go 概述与定价

> 来源：https://opencode.ai/docs/zh-cn/go/

---

## 定位

OpenCode Go 是 OpenCode 官方提供的低成本开源编程模型订阅服务。

## 定价

| 周期 | 价格 |
|:-----|:------|
| 首月 | **$5** |
| 次月起 | **$10/月** |

## 工作原理

Go 的工作方式与 OpenCode 中任何其他提供商（provider）一样：

1. 登录 [OpenCode Zen](https://opencode.ai/auth)，订阅 Go，复制 API 密钥
2. 在 TUI 中运行 `/connect` 命令，选择 `OpenCode Go`，粘贴 API 密钥
3. 在 TUI 中运行 `/models` 查看可用模型列表

> Go 是**完全可选**的，并非使用 OpenCode 所必需的条件。

## 背景

开源模型现在变得非常强大。在编程任务中，性能已接近专有模型，且便宜得多。但获得可靠、低延迟的访问可能很困难——各提供商在质量和可用性方面参差不齐。

OpenCode 团队做了三件事来解决：

1. **测试**了一组精选且与 OpenCode 配合良好的模型和提供商
2. **合作**确保这些服务正确提供
3. **基准测试**，得出推荐的模型和提供商列表

Go 让你能够以 $5 首月 / $10 每月的价格访问这些模型。

## 适用场景

- 不需要专有模型（Claude/GPT），开源模型够用
- 需要稳定、低延迟的模型访问
- 想要低成本订阅而非按量付费
- 国际用户（模型托管在 US、EU、Singapore）
