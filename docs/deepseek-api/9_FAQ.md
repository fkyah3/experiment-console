# 常见问题（FAQ）

> 来源：https://api-docs.deepseek.com/zh-cn/faq

## 账号问题

### 账号无法登录
收到"你的账号已被临时停用"，说明账号触发系统停用规则。可填写「账号停用申诉」表单，审核用时约 3 个工作日。

### 邮箱无法注册
暂不支持部分邮箱域名，建议使用 Gmail、Outlook、Hotmail、Yahoo 等国际通用邮箱。仍有问题请联系 `service@deepseek.com`。

### 注销账号
「个人信息」→「注销」。注销后 chat 平台所有对话记录永久清空，未消费余额视为自愿放弃。

## 实名认证

### 个人 vs 企业认证
两者权益和功能目前无区别。个人认证可变更为企业认证（「充值」→「对公汇款」→「企业实名认证」→「去变更」）。企业认证不可变更为个人认证。

### 实名认证失败
常见原因：身份证已绑定其他平台账号、连续输错次数过多被锁定。

## 财务问题

### 如何充值？
- **在线充值**：完成实名认证后，在「充值」页面使用支付宝/微信
- **对公汇款**：仅支持企业用户，完成企业实名认证后获取专属汇款账号

### 余额会过期吗？
充值余额永久有效。赠送余额有效期可在「账单」页面查看。

### 可以退款吗？
未消费金额支持退款。
- 在线支付：在「账单」→「退款管理」自助操作
- 对公转账：填写工单选择「申请退款」

### 如何开发票？
「账单」→「发票管理」。企业用户发票抬头需与实名认证信息一致，开票周期约 7 个工作日。

## API 调用问题

### 并发限制是多少？
当前阶段没有硬性并发上限。系统负载较高时，基于负载和短时历史用量的动态限流模型可能导致 503/429。暂不支持针对单个账号提高并发上限。

### 为什么 API 比网页端慢？
网页端默认流式输出（stream=true），字符逐块显示。API 默认非流式（stream=false），全部输出完才返回。开启 stream=true 可提升交互性。

### 为什么调用 API 时持续返回空行？
请求等待调度过程中，服务器持续返回空行（非流式）或 `: keep-alive`（流式）以保持 TCP 连接不超时。

### 是否支持 LangChain？
支持。DeepSeek 接口与 OpenAI 兼容，LangChain 可直接调用。

### 如何离线计算 Tokens 用量？
参考 Token 用量计算文档。
