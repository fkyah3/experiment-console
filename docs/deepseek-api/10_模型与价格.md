# 模型与价格

> 来源：https://api-docs.deepseek.com/zh-cn/quick_start/pricing
> 抓取时间：2026-05-06

## 模型参数

| | deepseek-v4-flash(1) | deepseek-v4-pro |
|:--|:--|:--|
| base_url (OpenAI) | `https://api.deepseek.com` | 相同 |
| base_url (Anthropic) | `https://api.deepseek.com/anthropic` | 相同 |
| 模型版本 | DeepSeek-V4-Flash | DeepSeek-V4-Pro |
| 思考模式 | 非思考与思考模式（默认） | 相同 |
| 上下文长度 | **1M** | **1M** |
| 输出长度 | 最大 **384K** | 最大 **384K** |
| JSON Output | 支持 | 支持 |
| Tool Calls | 支持 | 支持 |
| 对话前缀续写(Beta) | 支持 | 支持 |
| FIM 补全(Beta) | **仅非思考模式下支持** | **仅非思考模式下支持** |

> (1) `deepseek-chat` 和 `deepseek-reasoner` 将于 2026-07-24 弃用。当前分别映射到 `deepseek-v4-flash` 的非思考与思考模式。

## 价格

以**百万 tokens** 为单位计量。

| 项目 | deepseek-v4-flash | deepseek-v4-pro |
|:----|:-----------------|:----------------|
| 百万 tokens 输入（缓存命中）(2) | **0.02 元** | **0.1 元**（2.5折(3)） |
| 百万 tokens 输入（缓存未命中） | **1 元** | **12 元**（2.5折(3)） |
| 百万 tokens 输出 | **2 元** | **24 元**（2.5折(3)） |

- (2) 输入缓存命中价格已降至首发价格的 1/10（2026-04-26 起生效）
- (3) deepseek-v4-pro 当前 2.5 折，优惠期至 **2026-05-31 23:59**

## 扣费规则

```
扣减费用 = token 消耗量 × 模型单价
```

- 充值余额与赠送余额同时存在时，优先扣减赠送余额
- 产品价格可能变动，请定期查看最新价格信息
