⚙ **第一次用**
1. 下载解压，运行 ExperimentConsole.exe
2. 点右上角 ⚙ 设置 → 填入 DeepSeek API Key → 保存
3. 开始搭积木

🧱 **核心功能**
- 逐条增删改 system/user/assistant/tool 消息，thinking 开关随意切换
- 展开 assistant 查看推理过程（reasoning）和最终回答（content），颜色区分
- 批量实验模式：一键 40 轮，自动统计 reasoning 语言/token/耗时
- 完整请求体 + 响应体 + usage 日志，所有数据存 .md 可追溯
