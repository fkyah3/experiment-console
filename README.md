# Experiment Console（存档）

**DeepSeek API 消息积木实验台——项目已归档。**

此项目完成了它的使命：验证了 DeepSeek V4 在 Agent/工具调用场景下的 reasoning 语言行为，产出了可复现的实验数据和完整的分析报告。

## 项目结论

经过 500+ 次 API 实验、多层研讨、论文溯源和行业调研（Claude Code、Codex、DeepSeek-TUI、Vercel AI SDK），最终结论是：

> **多轮 tool call 是结构性问题，语言漂移只是症状之一。行业内没有标准解决方案。**

具体的工程方案已在移山项目中验证通过，40 轮、10/10、100% 中文 reasoning。但实现细节在项目完成前不公开。
